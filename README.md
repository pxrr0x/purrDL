### Description:
***purrDL is a command-line Python utility for downloading movies on Linux and Termux. It quickly scrapes torrent websites and fetches movies in just a few clicks.***

### Installation
#### Install dependencies (git, python):
 - For Linux
    - ```sudo apt update```
    - ```sudo apt install git python3 python3-pip -y```
 - For Android(Termux)
    - <a href="https://github.com/termux/termux-app/releases/tag/v0.118.1">Download and install Termux here</a>
    - ```apt update && apt upgrade -y```
    - ```apt install git python3 python3-pip -y```
  
#### Clone this repository
 - ```git clone https://github.com/pxrr0x/purrDL.git```

#### Enter the directory
 - ```cd purrDL```

#### Install additional dependencies and modules
 - ```python3 setup.py```

### Usage
```usage: python3 purrDL.py [movie name]```

1. Run the script.
2. Pick your movie from the list.
3. Select the resolution (if available).
4. Sit back and let it download.
